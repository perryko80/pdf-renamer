## Project
PDF renamer script — auto-renames financial research PDFs using a structured naming convention.

## Naming convention
YYMMDD-CompanyCode-CountryCode-Title.pdf

## Company codes
GS, Citi, CF40, DB, Nomura, JPM

## Country codes
TA = Taiwan

## Stack
Python, pdfplumber, Claude API

## Rules
- Never delete files without asking
- Always explain changes before making them
- Keep comments in English
